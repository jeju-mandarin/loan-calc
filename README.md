# 🏦 대출 이자 계산기

원리금 균등상환과 원금 균등상환 방식을 비교하여, 대출 총액, 상환 기간, 거치 기간에 따른 월별 상환 내역을 상세하게 계산해주는 웹 유틸리티입니다.

## ✨ 주요 기능 및 계산 로직

이 계산기는 사용자의 설정을 바탕으로 두 가지 상환 방식의 **총 이자 금액**과 **월별 상환 상세 일정**을 모두 제공합니다.

* **2가지 상환 방식 지원:**
    * **원리금 균등상환:** 매월 납부하는 원리금(원금+이자)이 동일합니다.
    * **원금 균등상환:** 매월 납부하는 원금은 동일하며, 이자가 점차 감소합니다.
* **원금 납부 유예기간 (거치기간) 반영:** 유예기간(개월)을 설정하여, 해당 기간 동안에는 원금 상환 없이 이자만 납부하도록 계산합니다.
* **상세 상환 일정표:**
    * 총 상환 기간(회차)별 납부 원금, 납부 이자, 월 납입액을 분리하여 표시합니다.
    * **상환 후 대출 잔액**을 매월 명확하게 표기하여 원금 감소 추이를 확인할 수 있습니다.
* **차액 비교:** 선택한 방식과 반대 방식의 **총 이자 차액**을 계산하여 결과를 즉시 비교할 수 있습니다.
* **주의 문구 포함:** 계산 결과의 한계를 고지하는 문구가 포함되어 있습니다.
